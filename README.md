# Crypto Protocol Analytics

## Overview

Crypto Protocol Analytics is a data analytics project focused on collecting, storing, and analyzing crypto protocol data across DeFi, Layer 1s, Layer 2s, and stablecoin ecosystems.

The goal of the project is to develop practical skills in:

- SQL
- PostgreSQL
- Data Modeling
- ETL Pipelines
- Data Analytics
- Data Visualization
- Crypto Research

## Planned Features

- Protocol fundamentals database
- TVL tracking
- Revenue and fee analytics
- Stablecoin flow analysis
- Sector performance dashboards
- Research note management
- Protocol comparison tools

## Tech Stack

- Python
- PostgreSQL
- SQL
- Pandas
- Streamlit
- Git/GitHub
